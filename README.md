# The Last Epiphany

> *"Não importa seu símbolo, se faz sentido para quem criou, faz sentido para todes!"*

**The Last Epiphany** é o site pessoal de **Larissa Ember** — uma espaço onde ela compartilha crônicas, artigos sobre programação e desenvolvimento web, seu perfil social e projetos autorais.

## 🌐 Acesse o site

[lari-ember.github.io/epiphany](https://lari-ember.github.io/epiphany/)

## 📄 Páginas

| Página | Descrição |
|---|---|
| [Home](index.html) | Página inicial com apresentação e destaques do site |
| [Crônicas](comicas.html) | Textos literários e criativos de Larissa |
| [Pokedex](projetos/pokedex.html) | Mini-projeto: busca de Pokémons pela PokéAPI |
| [Perfil Social](profile.html) | Informações pessoais e links para redes sociais |
